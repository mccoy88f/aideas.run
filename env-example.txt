# =================================
# CONFIGURAZIONE GOOGLE CLOUD CONSOLE
# =================================

# 1. Vai su https://console.cloud.google.com/
# 2. Crea un nuovo progetto o seleziona un progetto esistente
# 3. Vai su "APIs & Services" > "Credentials"
# 4. Clicca "Create Credentials" > "OAuth 2.0 Client ID"
# 5. Configura il consenso OAuth se richiesto
# 6. Seleziona "Web application" come tipo di applicazione
# 7. Aggiungi "Authorized JavaScript origins":
#    - Per sviluppo locale: http://localhost:3000
#    - Per produzione: https://aideas.run
# 8. Aggiungi "Authorized redirect URIs":
#    - Per sviluppo locale: http://localhost:3000/auth/google.html
#    - Per produzione: https://aideas.run/auth/google.html
# 9. Copia Client ID e Client Secret sopra

# =================================
# CONFIGURAZIONE GITHUB ACTIONS
# =================================

# Per il deployment su GitHub Pages, configurare le variabili negli Environment secrets:
# 1. Vai su Settings > Environments
# 2. Seleziona "github-pages" environment
# 3. Aggiungi le variabili in "Environment secrets":
#    - VITE_GOOGLE_CLIENT_ID
#    - VITE_GOOGLE_CLIENT_SECRET

# IMPORTANTE: Le variabili devono essere negli Environment secrets,
# NON nei Repository secrets!
